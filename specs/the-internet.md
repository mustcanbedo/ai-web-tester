# The Internet (Heroku) UI 组件测试

> 目标网址：https://the-internet.herokuapp.com/
> 无需登录

## 通用规则
- 每步操作后观察页面变化
- 注意页面加载时间，超过 5 秒视为异常
- 记录所有 console error 和 network error

---

### 1. 复选框交互
**步骤：**
1. 从首页点击 "Checkboxes" 链接
2. 观察页面上有 2 个复选框
3. 点击第一个复选框使其选中
4. 点击第二个复选框使其取消选中
5. 确认两个复选框的状态已正确切换

**结果预期：**
1. 第一个复选框从未选中变为选中
2. 第二个复选框从选中变为未选中

---

### 2. 下拉框选择
**步骤：**
1. 返回首页
2. 点击 "Dropdown" 链接
3. 在下拉框中选择 "Option 1"
4. 确认选中值为 "Option 1"
5. 再选择 "Option 2"
6. 确认选中值变为 "Option 2"

**结果预期：**
1. 下拉框能正确切换选项
2. 选中状态与显示一致

---

### 3. 登录表单验证
**步骤：**
1. 返回首页
2. 点击 "Form Authentication" 链接
3. 在用户名输入 "tomsmith"
4. 在密码输入 "SuperSecretPassword!"
5. 点击 "Login" 按钮
6. 确认登录成功提示出现
7. 点击 "Logout" 按钮
8. 确认返回登录页面

**结果预期：**
1. 正确凭据登录后显示成功消息
2. 登出后返回登录页

---

### 4. 悬停效果
**步骤：**
1. 返回首页
2. 点击 "Hovers" 链接
3. 鼠标悬停在第一个用户头像上
4. 确认出现用户名信息
5. 悬停在第二个头像上
6. 确认出现不同的用户名

**结果预期：**
1. 悬停时正确显示用户信息覆盖层
2. 每个头像对应不同的用户名
