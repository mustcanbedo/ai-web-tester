# UI 验证结果

## 界面元素确认

### 左侧面板（配置区）
- ✅ LLM 配置区：API Key、Base URL、模型名称输入框 + "测试连接"按钮
- ✅ 测试目标区：目标网址输入框
- ✅ 登录配置区：需要登录开关、登录方式切换（账号密码/手机验证码）
- ✅ 功能预期文档：在线编辑/上传文件切换，编辑器已加载示例内容
- ✅ 接口文档（可选）：提供接口文档开关，三层数据验证说明
- ✅ 操作按钮：加载示例、启动测试

### 右侧面板（监控区）
- ✅ LIVE 截图面板：每步操作自动推送当前页面截图（带 LIVE 指示器）
- ✅ 步骤截图列表：每步截图可点击放大
- ✅ 执行日志区：实时日志流
- ✅ 状态栏：步骤数、Bug 数、Token 用量（input/output 明细）
- ✅ 测试控制：暂停/恢复/终止按钮
- ✅ 人工输入弹窗：Agent 请求验证码等输入时弹出

### 测试完成后
- ✅ 查看报告按钮（Markdown 渲染）
- ✅ 播放视频按钮（Playwright 录制的 .webm）
- ✅ 从快照恢复按钮（异常终止时可用）

## 技术栈
- 前端：Vanilla JS 单页应用（index.html）
- 实时通信：SSE（Server-Sent Events，带 event ID 断连重连）
- 浏览器自动化：Playwright headless Chromium
- 页面观测：Accessibility Tree 快照（非截图/noVNC）
